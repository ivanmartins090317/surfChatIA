# Evidências — Testes Manuais

Screenshots dos casos de teste do relatório [`relatorio-testes-manuais.html`](../relatorio-testes-manuais.html).

## Nomenclatura

```
tc[número]-[descricao-curta].[ext]
```

Exemplos:
- `tc09-analise-link-pronta.jpeg`
- `tc15-upload-tres-fotos.png`
- `tc21-rls-usuario-b.jpeg`

## Formatos aceitos

`.jpeg`, `.jpg`, `.png`, `.webp`

## Como adicionar

1. Execute o TC no app (`npm run dev` → http://localhost:3000)
2. Capture a tela com o resultado
3. Salve nesta pasta com o nome do TC
4. Peça à AI para atualizar o relatório HTML (status + galeria)

## Privacidade

Não commitar screenshots com dados pessoais reais, tokens ou chaves de API visíveis.
